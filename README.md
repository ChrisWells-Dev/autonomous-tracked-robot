# Autonomous Tracked Robot

A ground-up autonomous tracked robot build. Pi4 compute hub, dual ESP32 motor controllers, lidar, and ROS2. Built to map and navigate indoor and eventually underground environments.

**Status: Active build - sections updated as hardware comes online.**
